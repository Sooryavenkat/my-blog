# Personal blog by Soorya — Security for PMs & AI Builders.

## Live at → sooryav.netlify.app

## What this is

Most security knowledge was written for developers and security engineers. AI changed who builds software — PMs, founders, marketers, and operations teams are now shipping real tools and making invisible security decisions nobody prepared them for.

This blog bridges that gap: security thinking for people who build with AI but weren't in the room when the security playbook was written.

## Topics covered:

1) Secure-by-design product practices for PMs
2) Hidden risks in AI-assisted and vibe-coded tools
3) Cybersecurity concepts made accessible to non-security builders
4) AI trends and tools worth paying attention to
